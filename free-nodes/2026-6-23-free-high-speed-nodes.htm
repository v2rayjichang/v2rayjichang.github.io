一、什么是Shadowrocket？

1.1 Shadowrocket简介

Shadowrocket 是一款适用于 iOS 设备（iPhone 和 iPad）的网络代理工具，支持 Shadowsocks、Vmess、Trojan 等多种代理协议。其主要作用是通过代理服务器访问互联网，提升网络自由度，同时增强安全性，防止数据泄露。

1.2 Shadowrocket的主要功能

Shadowrocket 之所以受到广大用户的喜爱，主要因为它具备以下特点：

  • 支持多种代理协议：Shadowsocks、Vmess、Trojan、HTTP、SOCKS5 等。

  • 自定义规则：可设置特定网站或应用是否走代理，支持 ACL 规则。

  • 实时流量监控：提供详细的流量统计，方便管理流量使用。

  • 高效稳定：支持 UDP 转发，保证流畅的网络连接体验。

  • DNS 解析功能：可自定义 DNS 设置，避免 DNS 污染，提高访问速度。

无论是访问受限网站，还是加密流量保护隐私，Shadowrocket 都是一个值得信赖的选择。

二、一键安装Shadowrocket的完整步骤

2.1 设备准备

在安装 Shadowrocket 之前，请确保你的设备符合以下条件：

✅ iOS 9.0 及以上（建议最新版本）
✅ 稳定的 WiFi 或移动网络
✅ 已登录 Apple ID（可访问 App Store）

2.2 下载安装Shadowrocket

  1. 打开 App Store，在搜索框输入 Shadowrocket。

  2. 点击购买（Shadowrocket 是付费应用），并完成支付。

  3. 等待安装完成，然后在桌面找到 Shadowrocket 的图标。

? Shadowrocket 仅在部分国家/地区的 App Store 提供，如果你的账户无法搜索到，可考虑更换 Apple ID 进行下载。

三、Shadowrocket的详细配置方法

Shadowrocket 支持 手动配置 和 一键导入配置 两种方式，下面将分别介绍。

3.1 一键导入配置（推荐新手使用）

步骤如下：

  1. 打开 Shadowrocket，点击右上角的 + 号。

  2. 选择 “从URL导入”。

  3. 输入配置 URL（VPN 提供商通常会提供），然后点击 确定。

  4. 代理配置会自动导入，返回主界面即可使用。

? 如果你从 VPN 提供商获取了配置 URL，这是最便捷的方式，不需要手动填写服务器信息。

3.2 手动配置 Shadowrocket（适合高级用户）

如果你想手动添加代理配置，请按照以下步骤操作：

  1. 打开 Shadowrocket，点击右上角的 + 号。

  2. 选择 “手动配置”，然后输入以下信息：

    • 服务器地址（VPN 提供商提供）

    • 端口号（一般为 443、8388 等）

    • 密码（获取的账户密码）

    • 加密方式（如 AES-256-GCM、ChaCha20 等）

  3. 选择合适的 代理协议（Shadowsocks、Vmess、Trojan）。

  4. 点击“保存”，然后返回主界面启动代理。

? 手动配置适合高级用户，可以自定义更多参数，但初学者建议使用 URL 一键导入方式。

3.3 连接代理

配置完成后，在主界面：

✅ 选择刚刚添加的代理服务器。
✅ 点击“连接”，等待连接成功（出现 VPN 标志）。

至此，你的设备已经成功连接到代理服务器，可以畅通无阻地访问互联网了！

FAQ

WinXray 支持 Trojan 协议吗？
WinXray 内核基于 Xray-core，因此原生支持 Trojan 协议。用户在导入配置时，只需选择 Trojan 类型节点即可，无需额外插件，非常适合需要 TLS 伪装的环境。
SSR 的协议插件如何应对封锁？
协议插件可隐藏真实通信特征，将流量伪装成正常 HTTP 或 TLS 数据。结合混淆方式使用，可以有效降低被 DPI 检测或封锁的风险，提高节点稳定性和长期可用性。
Potatso Lite 是否可导入节点二维码？
支持。用户可扫描节点二维码导入节点信息，也可手动输入节点参数。虽然不支持订阅自动更新，但对于轻量用户来说足够满足日常节点管理和流量分流需求。
Netch 支持游戏加速吗？
Netch 专为游戏加速设计，支持 UDP 转发和 TAP 驱动，可有效降低延迟。用户可以为特定游戏添加路由规则，实现分流加速，从而提升国际游戏体验。
Surfboard 支持按应用分流流量吗？
支持。Surfboard 可通过策略组和规则文件将不同应用的流量分配到指定节点，保证游戏、视频和浏览器流量走最优节点，提高访问速度和网络稳定性。
Matsuri 节点延迟排序如何优化网络流量？
Matsuri 会定期测速节点延迟，并结合策略组自动选择延迟最低节点。无需手动切换，确保网页、视频和游戏流量通过最优节点，提高访问速度和网络稳定性。
VLess 与 Reality 协议可以结合使用吗？
可以。VLess + Reality 是目前最受欢迎的组合之一。它在无证书情况下实现加密传输，同时具备隐蔽性与速度优势，非常适合需要安全访问的高级代理配置。
Hiddify Next 用户流量配额如何设置？
管理员可为每个用户设置带宽和流量上限，超出限制时可降速或阻止访问。确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性和使用效率。
Nekobox 能否使用订阅链接？
Nekobox 支持通过订阅 URL 导入节点。用户在应用中选择“添加订阅”，输入链接后系统会自动更新节点数据，并支持定期刷新，确保始终使用最新线路。
VLESS + Reality 与 Trojan 在性能上差异大吗？
两者性能相近。VLESS + Reality 伪装更自然，适合强封锁地区；Trojan 则依托标准 TLS 实现高速连接。选择取决于服务器支持与环境稳定性。
